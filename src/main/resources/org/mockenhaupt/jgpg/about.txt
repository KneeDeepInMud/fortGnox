Build: ${buildNumber}

Password manager based on GPG encryption.

    Mask 1st: First line in each file will be covered
    Mask in text: Passwords detected in text will be covered
    Squeeze text: Multiple blank lines in decoded text will be squeezed to one line
    URLs: Display URLs as symbolic links
    Open URLs: Open URLs in browser or copy them to clipboard on selection

The text area supports simple text search with F3 key navigation.

Masked passwords can be copied to clipboard while beeing covered from viewer
by using the respective buttons on the right side of the window.
